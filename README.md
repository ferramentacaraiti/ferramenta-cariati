# Ferramenta Cariati - sito statico

Questa cartella contiene una prima versione statica del sito, pronta per GitHub Pages.

## File principali

- `index.html`: pagina del sito
- `styles.css`: grafica e layout responsive
- `script.js`: menu mobile e anno automatico nel footer
- `logo.png`: logo usato nel sito
- `contenuti-riordinati.md`: testi riordinati per modifiche future

## Come pubblicarlo su GitHub Pages

1. Create un repository GitHub, per esempio `ferramenta-cariati`.
2. Caricate nella radice del repository questi file:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `logo.png`
   - `README.md`
3. In GitHub aprite `Settings`.
4. Entrate in `Pages`.
5. Alla voce `Build and deployment`, scegliete `Deploy from a branch`.
6. Selezionate il branch principale, di solito `main`, e la cartella `/root`.
7. Salvate.

Dopo qualche minuto il sito sara visibile all'indirizzo indicato da GitHub Pages.

## Cosa personalizzare dopo

- Aggiungere una foto reale dell'ingresso o dell'interno del negozio.
- Aggiungere email, partita IVA e link social, se disponibili.
- Collegare un dominio personalizzato, per esempio `ferramentacariati.it`.
